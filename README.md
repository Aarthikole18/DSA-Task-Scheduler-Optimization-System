# ⚡ Task Scheduler Optimization System (DSA + Streamlit)

A visually interactive **Task Scheduling Optimization System** built using Python, Streamlit, Pandas, and Plotly. It demonstrates how greedy algorithms can be used to simulate real-world scheduling problems like CPU scheduling, project task planning, and workflow optimization.

---

## 🚀 Live Features

- ➕ Add tasks dynamically using UI
- 🧠 Greedy priority-based scheduling algorithm
- ⏱ Deadline-aware execution system
- ❌ Automatic missed task detection
- 📊 Interactive execution timeline (Plotly chart)
- 📋 Live task table view
- 📌 KPI dashboard (Total / Scheduled / Missed tasks)
- 🎨 Clean modern UI using Streamlit

---

## 🎯 Problem Statement

Real-world systems (OS schedulers, project managers, cloud systems) need to execute tasks efficiently under constraints like limited time, deadlines, priority levels, and execution duration.

This project solves:
👉 How to schedule tasks optimally to maximize priority execution while minimizing missed deadlines?
